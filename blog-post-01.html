🔧 Problèmes résolus

  • Correction des ombres incorrectes sur Call of Duty®: Black Ops 7 (RX 5000/6000)
  • Correction de la corruption de texture dans Enshrouded (RX 7000/9000)
  • Correctif pour les crash Diablo 4 avec caractères non-anglais (RX 7000/9000)
  • Correction de l'affichage des lumières dans Microsoft Flight Simulator 2024 (RX 9000)
  • Crash de l'Unreal Engine 5.6 Editor avec Lumen HWRT corrigé (RX 5000/6000/7000/9000)
  • Corruption visuelle intermittente sur Chromium/Electron (RX 5000/6000)
  • Redimensionnement de fenêtre amélioré pour Vulkan API (RX 7000/9000)
  • Correction des coupures audio pendant la lecture vidéo (RX 9000)
  • Rendu des sprites corrigé dans Cassette Beasts (RX 7000/9000)
  • Artefacts TAA résolus dans Baldur's Gate 3 (RX 5000/6000)

📥 Installation

Pour les utilisateurs de carte RDNA, le driver peut être téléchargé directement depuis le site officiel AMD, ou via l'utilitaire Auto-detect and Install.

Conseil : Si vous mettez à jour depuis une ancienne version, utilisez l'AMD Cleanup Utility pour une installation propre.

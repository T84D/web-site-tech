Mise à jour AMD Adrenalin 26.5.1

06‑05‑2026

AMD Software: Adrenalin Edition 26.5.1

Nouvelle mise à jour disponible depuis le 6 mai 2026.

🎯 Points forts

  • Support de nouveaux jeux
  • Corrections de bugs signalés sur les Radeon RX 9000

🎮 Nouveau(x) jeu(x) supporté(s)

  • PRAGMATA
  • Honor of Kings: World
  • INDUSTRIA 2
  • Tides of Tomorrow
  • MONGIL: STAR DIVE

🔧 Problèmes résolus

  • Scintillement intermittent possible lors de la lecture de la carte "Raccoon City" dans Resident Evil Requiem sur les cartes Radeon™ RX 9000
  • Corruption possible dans God of War sur les cartes Radeon™ RX 9000

⚠️ Problèmes connus

  • Crash/timeout intermittent en jouant à Battlefield 6 sur Ryzen AI 9 HX 370. AMD travaille activement sur une résolution avec le développeur
  • Scintillement ou corruption de textures dans Battlefield 6 avec AMD Record and Stream sur certaines cartes graphiques
  • AMD FSR Upscaling et Frame Generation peuvent apparaître inactifs sur RX 9000 avec Battlefield 6
  • Crash/timeout intermittent en jouant à RoadCraft sur les cartes Radeon™ RX 9000
  • Corruption possible dans Satisfactory sur les cartes Radeon™ RX 9000

📥 Installation

Pour les utilisateurs de GPU RDNA, le pilote AMD Software: Adrenalin Edition 26.5.1 est téléchargeable depuis le site officiel AMD ou via l'utilitaire Auto-detect and Install Driver Update Option.

Conseil : utiliser l'AMD Cleanup Utility pour un nettoyage propre lors d'une mise à niveau.

Important : Ce pilote n'est pas prévu pour Apple Boot Camp. Les utilisateurs doivent contacter leur fabricant.

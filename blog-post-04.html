Mise à jour AMD Adrenalin 26.2.1

11‑02‑2026

AMD Software: Adrenalin Edition 26.2.1

Nouvelle mise à jour disponible depuis le 11 février 2026.

🎯 Nouvelles fonctionnalités

  • Optimisations continues pour la stabilité et la performance

🎮 Nouveaux jeux supportés

  • Yakuza Kiwami 3 & Dark Ties
  • Nioh 3

🔧 Problèmes résolus

  • Nuages corrompus dans Arc Raiders (RX 9000)
  • Crash intermittent dans The Finals avec ray tracing activé (RX 7000)

⚠️ Problèmes connus

  • Crash intermittent dans Cyberpunk 2077 avec Path Tracing activé
  • Crash intermittent dans Battlefield™ 6 sur Ryzen AI 9 HX 370
  • Crash intermittent dans Roblox Player lors du changement de tâche (RX 7000)
  • Scintillement ou corruption de texture dans Battlefield™ 6 avec Record and Stream
  • AMD FSR Upscaling et Frame Generation peuvent apparaître comme inactifs dans Yakuza Kiwami 3 & Dark Ties et Call of Duty®: Black Ops 7 (RX 9000)
  • AMD Install Manager ne peut pas installer les drivers optionnels (solution : télécharger depuis la page officielle)

📦 Compatibilité GPU

Cartes de bureau :

  • AMD Radeon™ RX 9070 / 9060 Series
  • AMD Radeon™ AI PRO R9700
  • AMD Radeon™ AI PRO R9700S
  • AMD Radeon™ AI PRO R9600D
  • AMD Radeon™ RX 7900/7800/7700/7650/7600 Series
  • AMD Radeon™ RX 6900/6800/6700/6600/6500/6400 Series
  • AMD Radeon™ RX 5700/5600/5500/5300 Series

Cartes portables :

  • AMD Radeon™ RX 7900M/7800M/7600M Series
  • AMD Radeon™ RX 6800M/6700M/6600M/6500M/6300M Series
  • AMD Radeon™ RX 5700M/5600M/5500M/5300M Series

🔧 Processeurs compatibles

Desktop :

  • AMD Ryzen™ Processors with Radeon™ Graphics
  • AMD Ryzen™ PRO Processors
  • AMD Athlon™ Processors with Radeon™ Graphics
  • AMD Athlon™ PRO Processors

Mobile :

  • AMD Ryzen™ AI Series Processors with Radeon™ Graphics
  • AMD Ryzen™ Processors with Radeon™ Graphics
  • AMD Ryzen™ PRO Processors
  • AMD Athlon™ Processors with Radeon™ Graphics
  • AMD Athlon™ PRO Processors

💻 Systèmes d'exploitation compatibles

  • Windows® 11 version 21H2 et ultérieur
  • Windows® 10 64-bit version 21H2 et ultérieur

📥 Installation

Pour les utilisateurs de carte RDNA, le driver AMD Software: Adrenalin Edition 26.2.1 peut être téléchargé directement depuis le site officiel AMD, ou via l'utilitaire Auto-detect and Install.

Conseil : Utilisez l'AMD Cleanup Utility lors d'une mise à niveau depuis une ancienne version pour une installation propre.

Important : Ne fonctionne pas sur Apple Boot Camp. Les utilisateurs Mac doivent contacter le fabricant de leur système.

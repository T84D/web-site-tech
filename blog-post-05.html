Mise à jour AMD Adrenalin 26.2.2

26‑02‑2026

AMD Software: Adrenalin Edition 26.2.2

Nouvelle mise à jour disponible depuis le 26 février 2026.

🎯 Points forts

  • Support de jeux additionnels
  • Corrections de bugs signalés sur les Radeon RX 7000

🎮 Nouveau(x) jeu(x) supporté(s)

  • Resident Evil Requiem
  • Marathon

🔧 Problèmes résolus

  • Crash ou timeout possible dans Roblox Player (Car Zone Racing & Drifting) lors du changement de tâche sur RX 7000
  • Scintillement ou corruption de textures avec des applications de bureau Windows quand Instant Replay et Record Desktop sont activés sur RX 7000

⚠️ Problèmes connus

  • Crash/timeout intermittent lors du chargement d'une partie sauvegardée dans Cyberpunk 2077 avec Path Tracing activé
  • Crash/timeout intermittent en jouant à Battlefield 6 sur Ryzen AI 9 HX 370
  • Scintillement ou corruption de textures dans Battlefield 6 avec AMD Record and Stream
  • AMD FSR Upscaling et Frame Generation peuvent apparaître inactifs sur RX 9000 avec certains jeux (Yakuza Kiwami 3 & Dark Ties, Call of Duty: Black Ops 7, Resident Evil Requiem)

📦 Compatibilité GPU

Cartes de bureau :

  • AMD Radeon™ RX 9070 / 9060 Series
  • AMD Radeon™ AI PRO R9700
  • AMD Radeon™ AI PRO R9700S
  • AMD Radeon™ AI PRO R9600D
  • AMD Radeon™ RX 7900/7800/7700/7650/7600 Series
  • AMD Radeon™ RX 6900/6800/6700/6600/6500/6400 Series
  • AMD Radeon™ RX 5700/5600/5500/5300 Series

Cartes portables :

  • AMD Radeon™ RX 7900M/7800M/7600M Series
  • AMD Radeon™ RX 6800M/6700M/6600M/6500M/6300M Series
  • AMD Radeon™ RX 5700M/5600M/5500M/5300M Series

🔧 Processeurs compatibles

Desktop :

  • AMD Ryzen™ Processors with Radeon™ Graphics
  • AMD Ryzen™ PRO Processors
  • AMD Athlon™ Processors with Radeon™ Graphics
  • AMD Athlon™ PRO Processors

Mobile :

  • AMD Ryzen™ AI Series Processors with Radeon™ Graphics
  • AMD Ryzen™ Processors with Radeon™ Graphics
  • AMD Ryzen™ PRO Processors
  • AMD Athlon™ Processors with Radeon™ Graphics
  • AMD Athlon™ PRO Processors

💻 Systèmes d'exploitation compatibles

  • Windows® 11 version 21H2 et ultérieur
  • Windows® 10 64-bit version 21H2 et ultérieur

📥 Installation

Pour les utilisateurs de GPU RDNA, le pilote AMD Software: Adrenalin Edition 26.2.2 est téléchargeable depuis le site officiel AMD ou via l'utilitaire Auto-detect and Install Driver Update Option.

Conseil : utiliser l'AMD Cleanup Utility pour un nettoyage propre lors d'une mise à niveau.

Important : Ce pilote n'est pas prévu pour Apple Boot Camp. Les utilisateurs doivent contacter leur fabricant.
